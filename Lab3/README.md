# 問題 - LCM文字走馬控制 

## 請設計可左/右位移之文字走馬控制程式。設計規格：

一開始，畫面清空(clear())，顯示cursor在左上角(第一列最左邊)。
-  以push button 1 控制第一組文字顯示在第一列，文字長度40。
- 以push button A 控制第二組文字顯示在第二列，文字長度40。 
- 以push button 4 控制文字左移。 
- 以push button D 控制文字右移。 
- 以push button BTN1 控制Clear Display。
- 以push button BTN2 控制Return Home。
- 以push button BTN4 控制Cursor on/off。
